# Mizizi Youth Organization Website

Final GitHub Pages package with:

- Mizizi Youth Organization logo in the header
- Updated profile-based content
- Program pages and program images
- Partner logo horizontal scrolling slider
- Gallery and multi-page navigation

## Upload to GitHub Pages

Upload the contents of this folder to the root of the GitHub repository, then commit.

Required root structure:

```txt
index.html
about.html
programs.html
gallery.html
partners.html
contact.html
css/
js/
assets/
```

Do not upload the ZIP file itself to GitHub Pages. Extract it first, then upload the contents.
